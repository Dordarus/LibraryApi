Authentication

Authentication / authenticate

POST auth/login
Login user and return auth token once user is authenticated

###Example cURL

curl \
-d email=[email] \
-d password=[password] \
http://localhost:3000/auth/login

Examples

{
  "auth_token":"eyJhbGciOiJIUzI1NiJ9.eyJ1c2VyX2lkIjoxLCJleHAiOjE1MTkxMzA3MzF9.d3iMiRwoqE1P33VSYV6iuNpORQ58pxJyDD0H2DvLNOc"
}

Params

  • email : required [

    Must be a String

    ]

    Email nessessary for signup

  • password : required [

    Must be a String

    ]

    Password nessessary for signup

Users

Users / create

POST /signup
Create a user and return authenticated token upon signup

###Example cURL

curl -X POST \
-d name=[name]
-d email=[email] \
-d password=[password] \
-d password=[password_confirmation] \
http://localhost:3000/signup

Examples

{
  "auth_token": "eyJhbGciOiJIUzI1NiJ9.eyJ1c2VyX2lkIjoyLCJleHAiOjE1MTkxMzEzNTN9.mHa431RKg7hLPKoX7z2V8ZsyBflheBoLdpbsrIIYUeg", 
  "message": "Account created successfully"
}

Params

  • user : optional [

    Must be a Hash

    ]

    User information

    • name : required [

      Must be a String

      ]

      Full name of user

    • email : required [

      Must be a String

      ]

      Email nessessary for signup

    • password : required [

      Must be a String

      ]

      Password nessessary for signup

    • password_confirmation : required [

      Must be a String

      ]

      Password confirmnation
