Superusefull Library Api

Resources

Authentication

Resource Description
POST auth/login Login user and return auth token once user is authenticated

Authors

Resource Description
GET /v1/authors Show all authors
GET /v1/authors/:id Show specific author with books
PATCH /v1/authors/:id Update specific author
POST /v1/authors Create a author
DELETE /v1/authors/:id Delete specific author

Books

Resource Description
GET /v1/books Show all books
GET /v1/books/:id Show specific book
POST /v1/books Create a book
PATCH /v1/books/:id Update specific book
DELETE /v1/books/:id Delete specific book

Users

Resource Description
POST /signup Create a user and return authenticated token upon signup
