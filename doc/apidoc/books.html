POST /v1/books
Create a book

###Example cURL

curl -X POST \
-H 'Authorization: [auth_token]' \
-d 'book[title]=[value]' \
-d 'book[genre]=[value]' \
-d 'book[year]=[value]' \
-d 'book[author_id]=[value]' \
http://localhost:3000/v1/books
